# Tubes1-ML-FFNN-G60
Repository ini dibuat untuk memenuhi Tugas Besar 1 Pembelajaran Mesin 2025/2026


### How to Run
#### 1. Create Environment
`bash
    python -m venv .venv
`

#### 2. Activate Environment
`bash
    # windows
    venv\Scripts\activate

    # mac/linux
    source .venv/bin/activate
`

#### 3. Install Requirements
`bash
    pip install -r requirements.txt
`
